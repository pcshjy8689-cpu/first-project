# 커피 주문 앱 Blueprint

## 개요

이 문서는 현대적인 웹 표준을 사용하여 구축된 세련된 커피 주문 애플리케이션의 디자인과 구현을 간략하게 설명합니다. 이 애플리케이션은 모바일 우선 접근 방식을 채택하고, 재사용 가능한 UI를 위해 웹 구성 요소를 활용합니다.

## 프로젝트 개요

### 디자인 및 스타일링

*   **감성:** 전문적이면서도 아늑한 느낌을 주기 위해 스타벅스와 감성적인 카페의 미학을 혼합합니다.
*   **색상 팔레트:** 따뜻한 나무 톤을 기본으로 하여 편안한 분위기를 조성합니다.
*   **타이포그래피:** 가독성이 좋고 심미적으로 만족스러운 한글 글꼴을 사용합니다.
*   **UI:** 메뉴 항목을 위한 카드 기반 인터페이스를 사용합니다.
*   **레이아웃:** 모바일 화면에 최적화된 반응형 디자인을 사용하며, 주문 버튼은 화면 하단에 고정됩니다.

### 기능

*   **메뉴:** 아메리카노, 카페라떼, 바닐라라떼, 그리고 다양한 전통차와 에이드, 스무디를 포함한 동적 메뉴 시스템. 최근 '잉글리쉬 블랙퍼스트' 티가 추가되었습니다.
*   **꼬마 바리스타 추천:** 사용자의 기분에 맞는 메뉴를 추천해주는 마스코트 캐릭터의 랜덤 팁 기능.
*   **웹 구성 요소:** 각 메뉴 항목은 캡슐화와 재사용성을 위해 동적으로 생성되는 카드 UI로 구현됩니다.
*   **주문:** 사용자는 메뉴 항목을 장바구니에 추가할 수 있습니다.
*   **주문 요약:** 하단 고정 버튼에 현재 주문의 총액과 수량을 표시합니다.

## 현재 계획

*   **목표:** 한국적인 감성의 모바일 커피 주문 웹사이트를 제작합니다.
*   **단계:**
    1.  **`blueprint.md` 업데이트:** 프로젝트의 목적, 디자인, 기능 및 현재 개발 계획을 한국어 버전으로 새로 문서화합니다.
    2.  **`index.html` 수정:**
        *   페이지 언어를 한국어 (`ko`)로 설정하고 제목을 "커피 주문"으로 변경합니다.
        *   헤더와 메인 콘텐츠 영역을 한국어에 맞게 수정합니다.
        *   화면 하단에 주문 정보를 표시하고 주문을 확정할 수 있는 고정 푸터(`order-footer`)를 추가합니다.
    3.  **`style.css` 수정:**
        *   `Noto Sans KR` 한글 글꼴을 가져옵니다.
        *   따뜻한 우드톤의 배경과 카드 UI에 어울리는 색상 팔레트를 새로 정의합니다. (`--wood-bg`, `--card-bg`, `--primary-text`, `--accent-color`)
        *   모바일 레이아웃에 맞게 `main`과 메뉴 카드(`coffee-item`) 스타일을 재정의합니다.
        *   화면 하단에 주문 버튼이 고정되도록 스타일을 추가합니다.
    4.  **`main.js` 수정:**
        *   커피 메뉴를 '아메리카노', '카페라떼', '바닐라라떼'로 업데이트하고, 각 메뉴에 어울리는 이미지 URL을 추가합니다.
        *   `<coffee-item>` 웹 구성 요소가 이미지를 표시할 수 있도록 구조를 변경하고, 내부 텍스트를 한국어로 수정합니다.
        *   장바구니 기능을 수정하여 하단의 고정 주문 버튼(`order-footer`)에 총 수량과 가격이 실시간으로 업데이트되도록 합니다. 가격은 원화(KRW) 형식으로 표시합니다.
